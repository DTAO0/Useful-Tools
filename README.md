一、C基础

1.# 预处理前解
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
    1. 符号 # 表示这是一个预处理指令，告诉编译器在编译源代码之前，要先执行一些操作。编译器在编译过程开始之前的预处理阶段会处理这些指令。
    
    2. stdio 是 “standard input & output” 的缩写，包含了编译器理解 printf() 以及其它输入 / 输出函数所需要的信息。
